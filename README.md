# دليل نظام تصنيف مخاطر البضائع (WCO) - عرض تفاعلي صوتي

## نبذة عن المشروع

موقع إلكتروني تفاعلي شامل يقدم **دليل نظام تصنيف مخاطر البضائع (WCO)** بصيغة عرض تقديمي حديث مزود بـ:

- **31 شريحة احترافية** تغطي كافة جوانب إدارة المخاطر الجمركية
- **شرح صوتي مفصل** لكل شريحة باللغة العربية الفصحى
- **واجهة تفاعلية متقدمة** للتحكم الكامل بالعرض
- **تصميم استجابي** يعمل على جميع الأجهزة (الحاسوب، الجوال، التابلت)

## محتوى العرض

### المحاور الرئيسية:

1. **المقدمة والأهداف الاستراتيجية**
2. **المرجعيات الدولية** (اتفاقية كيوتو، WTO TFA، إطار SAFE)
3. **المفاهيم الأساسية** في إدارة المخاطر الجمركية
4. **نموذج نضج إدارة المخاطر** ومستويات التطور
5. **الهيكل التنظيمي** والوحدات الوظيفية
6. **مصفوفة المسؤوليات (RACI)**
7. **الإطار التشريعي والقانوني**
8. **حماية البيانات والخصوصية**
9. **الأخلاقيات والإنصاف**
10. **المسارات الأربعة للاستهداف**:
    - المسار الأخضر (تسهيل كامل)
    - المسار الأصفر (تدقيق مستندي)
    - المسار الأحمر (فحص شامل)
    - المسار الأزرق (التدقيق اللاحق)
11. **مصادر البيانات** وتكامل الأنظمة
12. **جودة البيانات** والمعايير الموحدة
13. **دورة حياة تحليل البيانات**
14. **نظام النقاط (Scoring)** والاستهداف
15. **بناء ملفات المخاطر (Profiling)**
16. **التكامل مع التدقيق اللاحق (PCA)**
17. **العمليات اليومية** في مركز الاستهداف
18. **نظام التغذية الراجعة** والتحسين المستمر
19. **مؤشرات الأداء الرئيسية (KPIs)**
20. **التدريب وبناء القدرات البشرية**
21. **التعاون والتنسيق بين الجهات (IBM)**
22. **الجدول الزمني للتنفيذ** (12 شهراً)
23. **التحديات والحلول**
24. **الخاتمة والخطوات القادمة**

## كيفية الاستخدام

### الوصول للموقع:
افتح ملف `index.html` أو انتقل إلى الرابط الدائم للموقع.

### التحكم بالعرض:

**أزرار التحكم:**
- **تشغيل**: بدء تشغيل الشرح الصوتي للشريحة الحالية
- **إيقاف**: إيقاف الصوت مؤقتاً
- **إيقاف كامل**: إيقاف الصوت وإعادة تعيينه
- **السابق/التالي**: التنقل بين الشرائح
- **ملء الشاشة**: عرض العرض بملء الشاشة

**لوحة المفاتيح:**
- **السهم الأيمن**: الذهاب للشريحة السابقة
- **السهم الأيسر**: الذهاب للشريحة التالية
- **مفتاح المسافة**: تشغيل/إيقاف الصوت

**شريط الصوت:**
- تحكم مباشر بمستوى الصوت
- عرض الوقت الحالي والمدة الكلية

## الميزات الرئيسية

✅ **تفاعلية كاملة**: تحكم كامل بالصوت والشرائح
✅ **انتقال تلقائي**: الانتقال التلقائي للشريحة التالية عند انتهاء الشرح
✅ **استجابة عالية**: تصميم يتكيف مع جميع أحجام الشاشات
✅ **أداء سريع**: تحميل سريع للملفات الصوتية والشرائح
✅ **سهولة الاستخدام**: واجهة بديهية وسهلة الفهم
✅ **دعم اللغة العربية**: واجهة كاملة باللغة العربية من اليمين لليسار

## المتطلبات التقنية

- متصفح ويب حديث (Chrome, Firefox, Safari, Edge)
- اتصال إنترنت لتحميل الملفات الصوتية
- دعم HTML5 و CSS3 و JavaScript
